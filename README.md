# 🧠 HackerEarth Solutions

A curated collection of **competitive programming solutions** solved on [HackerEarth](https://www.hackerearth.com/), written in **C++**.

---

## 📂 Repository Structure

All solutions are stored at the root level, each in its own `.cpp` file named after the problem title.

```

├── Count_Divisors.cpp
├── Longest_Paths_in_Tree.cpp
├── Modulo_Fermat_s_Theorem.cpp
└── ... (100+ problems)
```

## 🚀 How to Run

### Prerequisites
- A C++ compiler: `g++` (GCC) or `clang++`

### Compile & Execute

```bash
# Compile
g++ -std=c++17 -O2 -o solution <ProblemName>.cpp

# Run
./solution
```

### Example

```bash
g++ -std=c++17 -O2 -o sol Count_Divisors.cpp
./sol
```
